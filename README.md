# CARA

CARA is a Rust benchmark playground for routing policies in a distributed ANN routing setting. The project compares Random, Round Robin, Least Loaded, Leader Only, and CARA routing under the same workloads and emits a final report with CSV data and plots.

## What the report contains

The benchmark sweep writes its outputs to `benchmark_output/`:

- `benchmark_results.csv` - per-run benchmark metrics
- `ranking.csv` - average algorithm rank across all sweeps
- `ranking.png` - visual ranking chart
- `avg_latency_nodes_*.png` - latency plots for each node count

## Current result

Based on the generated ranking report, CARA is the best overall algorithm by average rank.

## Build and run

```bash
cargo build
cargo run --release
```

## Notes

- The sweep covers node counts from 4 to 20.
- The workload sweep covers request counts from 1,000 to 10,000.
- The benchmark output can be regenerated by running the binary again.
